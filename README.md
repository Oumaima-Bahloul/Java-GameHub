# Java-GameHub
A Game Hub built in Java using Swing, featuring a collection of classic games accessible from a single arcade-themed launcher.

## 📹 Demo
 
[![GameHub Demo](https://img.youtube.com/vi/yt1J3R0MCng/0.jpg)](https://www.youtube.com/watch?v=yt1J3R0MCng)
 


## 🕹️ Games Included
 
| Game | Description |
|------|-------------|
| 🟡 **Pac-Man** | Navigate the maze, eat pellets and avoid ghosts |
| 🐦 **Flappy Bird** | Tap to fly through pipes and beat your high score |
| 🐍 **Snake** | Eat food, grow longer, don't hit the walls |
| 🃏 **Black Jack** | Classic card game against the dealer |
| ✏️ **Sudoku** | Randomly generated puzzles every time you play |
| ❌ **Tic-Tac-Toe** | Two-player classic on a 3x3 grid |
| ✂️ **Rock Paper Scissors** | Play against the computer |

## 🚀 Getting Started
 
### Prerequisites
- Java JDK 17 or higher
- IntelliJ IDEA (recommended)
### Running the Project
1. Clone the repository
```bash
git clone https://github.com/Oumaima-Bahloul/GameHub-Java.git
```
2. Open the project in IntelliJ IDEA
3. Mark `src/resources` as **Resources Root**
4. Run `Main.java`
